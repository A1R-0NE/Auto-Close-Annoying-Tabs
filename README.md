# Auto Close Blocked Tabs

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

I'm done with ads or sites opening new tab redirects and manually closing them. So I made a simple and lightweight browser extension that **automatically** closes annoying and unwanted redirect tabs based on a personal blocklist.
## üì± [Screenshot](https://imgur.com/a/8vwcstT)


## üöÄ Features

* **Auto-Close:** Instantly closes tabs whose URL or Title matches a keyword on your blocklist.
* **Custom List:** Easily add or remove keywords and sites from your list.
* **Quick Block:** A "Block This Site" button to quickly add the current website's domain to your list.
* **Syncs:** Your blocklist is synced across your browsers using `chrome.storage.sync`.
* **Lightweight:** Runs efficiently in the background without slowing you down.

## üõ†Ô∏è Installation

Since this extension is not on the Chrome Web Store, you can install it manually in developer mode.

1.  **Download:** Download this repository as a ZIP file and unzip it, or clone it to your computer.
2.  **Open Extensions:** Open your browser's extensions page.
    * **Chrome:** Go to `chrome://extensions`
    * **Brave:** Go to `brave://extensions`
3.  **Enable Developer Mode:** Find the "Developer mode" toggle (usually in the top-right corner) and turn it **ON**.
4.  **Load Extension:** Click the **"Load unpacked"** button that appears.
5.  **Select Folder:** Select the folder where you unzipped the extension files.

The extension is now installed and active! <br>
And you should see the default blocklist I've created.

## üìñ How to Use

1.  Click the extension's icon in your browser toolbar to open the popup.
2.  **To block your current site:** Click the **"Block This Site"** button. The page will reload and instantly close.
3.  **To add a custom keyword:** Type a URL, domain, or title keyword (like "Casino" or "Betting") [case sensitive] into the text box and click **"Add"**.
4.  **To remove a keyword:** Click the `√ó` button next to any item in the list.
